---
 
Welcome.
 
 
 
Usage
---
 
Clone git project
 
```
./install.sh [GIT_PROJECT]
```
 
Example: ./install.sh git@github.com:jekyll/jekyll.git 
 
Run docker services
---
 
```
./run.sh [PROJECT]
```
 
Example: ./run.sh jekyll.local 
 
Destroy docker service
---
 
```
./destroy.sh
```
